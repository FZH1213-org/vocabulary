# 词汇复习表格 - 微信小程序

基于HTML词汇复习表格开发的微信小程序版本。

## 功能特点

- **词汇展示**：显示中文、音标、例句挖空
- **默写检查**：输入英文单词，点击检查核对拼写
- **批量操作**：检查全部、显示答案、清空重做
- **章节导航**：快速跳转到不同Story章节
- **发音功能**：点击中文卡片播放英文发音（使用有道词典API）
- **正确率统计**：显示已测数量和正确率

## 数据来源

词汇数据来自《马里亚纳海沟的秘密》词汇复习表，包含531个重点考研词汇，按10个故事章节排列。

## 使用说明

1. **导入项目**：使用微信开发者工具导入 `vocabulary-miniapp` 文件夹
2. **配置AppID**：在 `project.config.json` 中修改 `appid` 为你自己的小程序AppID
3. **预览/发布**：点击开发者工具中的"预览"或"上传"按钮

## 交互操作

- 点击中文词汇旁边的🔊图标可以播放英文发音
- 在输入框输入英文单词后点击"检查"按钮核对
- 点击"检查全部"一次性检查所有已填写的单词
- 点击"显示答案"查看所有正确答案
- 点击"清空重做"清除所有输入重新开始
- 使用章节导航快速跳转到不同章节

## 注意事项

- 发音功能使用有道词典在线API，需要网络连接
- 如需离线使用发音功能，建议集成微信小程序的语音合成API或本地音频文件
